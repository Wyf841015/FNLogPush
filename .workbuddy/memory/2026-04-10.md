# 2026-04-10 工作日志

## 修复侧边栏和移动端底部导航栏点击无响应问题

### 问题描述
Web侧边栏菜单和移动端底部菜单点击没有反应。

### 问题原因
1. `switchNavPanel` 函数使用了错误的选择器 `.nav-panel`，但实际面板使用的是 `.config-panel` 类
2. 面板ID格式为 `panel-status`、`panel-basic` 等，但函数直接使用 `target` 作为ID查找
3. `syncNavActive` 函数没有同步更新移动端底部导航栏按钮的 active 状态

### 修复内容
文件: `cmd/logmonitor/src/static/js/sidebar.js`

1. 修改 `switchNavPanel` 函数:
   - 选择器从 `.nav-panel` 改为 `.config-panel`
   - 面板ID查找从 `document.getElementById(target)` 改为 `document.getElementById('panel-' + target)`
   - 添加滚动到页面顶部功能

2. 修改 `syncNavActive` 函数:
   - 添加移动端底部导航栏按钮的 active 状态同步
   - 使用 `data-target` 属性匹配对应的移动端导航按钮
